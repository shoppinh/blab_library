# Trả về URL nhà cung cấp hiện tại
```javascript
web3.currentProvider
web3.eth.currentProvider
web3.shh.currentProvider
web3.bzz.currentProvider
...
```
Sẽ trả lại nhà cung cấp hiện tại, nếu không `null`.
### Trả lại
`Object`: Nhà cung cấp hiện tại hoặc` null`.
