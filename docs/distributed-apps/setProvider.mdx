# Thiết lập nhà cung cấp module

```javascript
web3.bzz.setProvider(myProvider)
```
Sẽ thay đổi nhà cung cấp cho nút của nó
:::note
Khi được gọi trên gói ô `Web3`, nó cũng sẽ đặt nhà cung cấp cho tất cả các mô -đun phụ` Web3.eth`, `Web3.shh`, v.v.
:::

### Thông số
1. `Object` - `myProvider`: một nhà cung cấp hợp lệ.
### Trả lại
`Boolean`
### Ví dụ
```javascript
var Bzz = require('web3-bzz');
var bzz = new Bzz('http://localhost:8500');

// change provider
bzz.setProvider('http://swarm-gateways.net');

```
