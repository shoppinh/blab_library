# Upload các tệp, thư mục hoặc dữ liệu thô trên các ứng dụng phân tán (swam)
```javascript
web3.bzz.upload(mixed)
```
Tải lên các tập tin, thư mục hoặc dữ liệu thô lên bầy.
### Parameters
1. `mixed` - `String|Buffer|Uint8Array|Object`:** Dữ liệu để tải lên, có thể là nội dung tệp, bộ đệm tệp/uint8array, nhiều tệp hoặc thư mục hoặc tệp (chỉ trong Node.js).Các loại sau được cho phép: **
+ `String|Buffer|Uint8Array`: Một nội dung tệp, tệp uint8array hoặc bộ đệm để tải lên hoặc:
+ `** Đối tượng **`:
    + ** Nhiều giá trị chính cho các tệp và thư mục.Các đường dẫn sẽ được giữ giống nhau: **
        + ** phím phải là đường dẫn tệp hoặc tên, ví dụ:`**"/foo.txt "**` và giá trị của nó là một đối tượng có: **
            + `type`: loại mime của tệp, ví dụ:`" Text/html "`.
            + `Data`: Một nội dung tệp, tệp UINT8Array hoặc bộ đệm để tải lên.
    + ** Tải lên tệp hoặc thư mục từ đĩa trong Node.js.Yêu cầu và đối tượng với các thuộc tính sau: **
        + `đường dẫn`: Đường dẫn đến tệp hoặc thư mục.
        + `Kind`: loại của nguồn` "Thư mục" `,` "Tệp" `hoặc` "Dữ liệu" `.
        + `defaultFile` (tùy chọn): Đường dẫn của DefaultFileFile khi` "loại" `:` "thư mục" `, ví dụ:`" /index.html "`.
    + ** Tải lên tệp hoặc thư mục trong trình duyệt.Yêu cầu và đối tượng với các thuộc tính sau: **
        + `pick`: Trình chọn tệp để khởi chạy.Có thể là `" tệp "`, `" thư mục "` hoặc `" dữ liệu "`.
### Trả lại
`Promise` returning `String`:Trả về nội dung băm của bản kê khai.
### Ví dụ
```javascript
var bzz = web3.bzz;

// raw data
bzz.upload("test file").then(function(hash) {
    console.log("Uploaded file. Address:", hash);
})

// raw directory
var dir = {
    "/foo.txt": {type: "text/plain", data: "sample file"},
    "/bar.txt": {type: "text/plain", data: "another file"}
};
bzz.upload(dir).then(function(hash) {
    console.log("Uploaded directory. Address:", hash);
});

// upload from disk in node.js
bzz.upload({
    path: "/path/to/thing",      // path to data / file / directory
    kind: "directory",           // could also be "file" or "data"
    defaultFile: "/index.html"   // optional, and only for kind === "directory"
})
.then(console.log)
.catch(console.log);

// upload from disk in the browser
bzz.upload({pick: "file"}) // could also be "directory" or "data"
.then(console.log);
```
