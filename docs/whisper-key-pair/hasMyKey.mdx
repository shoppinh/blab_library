# Kiểm tra khóa riêng tương ứng với ID đã cho hay không

```javascript
web3.shh.hasSymKey(id, [callback])
```
Kiểm tra xem có khóa đối xứng được lưu trữ với ID đã cho không.
### Tham số
1. `String` - ID cặp khóa, được trả về bởi các chức năng sáng tạo (shh.newsymkey, shh.addsymkey hoặc shh.generatesymkeyfrompassword).
2. `function` - (tùy chọn) gọi lại tùy chọn, trả về một đối tượng lỗi làm tham số đầu tiên và kết quả là thứ hai.
### Trả lại
`Boolean` -` true 'nếu khóa đối xứng tồn tại trong nút, sai nếu không.Lỗi về thất bại.
### Ví dụ
```javascript
web3.shh.hasSymKey('f6dcf21ed6a17bd78d8c4c63195ab997b3b65ea683705501eae82d32667adc92')
.then(console.log);
> true
```
