# Tạo cặp khóa từ mật khẩu
```javascript
createKeyPairFromPassword(password)
```
Lưu trữ một cặp khóa có nguồn gốc từ khóa riêng và trả về ID của nó.
### Thông số
1. `String` - Mật khẩu.
### Trả lại
`Object` - Thông tin cặp khóa

###  Ví dụ
```javascript
createKeyPairFromPassword('abcdef123')
.then(console.log);
```
