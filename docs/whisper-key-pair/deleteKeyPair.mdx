# Xóa cặp khóa riêng và khóa công khai
```javascript
web3.shh.deleteKeyPair(id, [callback])
```
Xóa khóa chỉ định nếu nó tồn tại.
### Thông số
1. `String` - ID cặp khóa, được trả về bởi các chức năng tạo (shh.newKeypair và shh.addprivateKey).
2. `function` - (tùy chọn) gọi lại tùy chọn, trả về một đối tượng lỗi làm tham số đầu tiên và kết quả là thứ hai.
### Trả lại
`Boolean` -` True 'thành công, lỗi về thất bại.

### Ví dụ
```javascript
web3.shh.deleteKeyPair('3e22b9ffc2387e18636e0a3d0c56b023264c16e78a2adcba1303cefc685e610f')
.then(console.log);
> true
`
```
