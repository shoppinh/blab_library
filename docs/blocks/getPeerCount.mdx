# Lấy số lượng block đã kết nối với nhau

```javascript
web3.eth.net.getPeerCount([callback])
web3.bzz.net.getPeerCount([callback])
web3.shh.net.getPeerCount([callback])
```
Nhận số lượng đồng nghiệp kết nối với.

### Thông số
không có

### Trả lại
`Promise` trả lại `Number`

### Ví dụ
```javascript
web3.eth.net.getPeerCount()
.then(console.log);
> 25
```
