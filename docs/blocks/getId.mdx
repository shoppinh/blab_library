# Nhận ID mạng hiện tại

```javascript
web3.eth.net.getId([callback])
web3.bzz.net.getId([callback])
web3.shh.net.getId([callback])
```
Nhận ID mạng hiện tại.
## Thông số
none
### Trả lại
`Promise` returns `Number`: ID mạng.
### Ví dụ
```javascript
web3.eth.net.getId()
.then(console.log);
> 1
```
