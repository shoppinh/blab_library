# Xem phiên bản whisper đang chạy

```javascript
web3.shh.getVersion([callback])
```
Trả về phiên bản của The Running Whisper.
### Thông số
`Function` - (optional) Callback tùy chọn, trả về một đối tượng lỗi làm tham số đầu tiên và kết quả là thứ hai.
### Trả lại
`String` - Phiên bản của tiếng thì thầm hiện tại đang chạy.

### Ví dụ
```javascript
web3.shh.getVersion()
.then(console.log);
> "5.0"
```
