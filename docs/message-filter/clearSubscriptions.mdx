# Đặt lại đăng ký
```javascript
web3.shh.clearSubscriptions()
```
Đặt lại đăng ký.
:::note

Điều này sẽ không đặt lại đăng ký từ các gói khác như Web3-ETH, vì họ sử dụng RequestManager của riêng họ.
:::
### Thông số
`Boolean`: Nếu đúng, nó giữ cho đăng ký "đồng bộ hóa".
### Trả lại
`Boolean`

### Ví dụ
```javascript
web3.shh.subscribe('messages', {...} ,function(){ ... });

...

web3.shh.clearSubscriptions();
```
