# Ký tin nhắn
```javascript
signMessage(privateKey, message)
```
Lấy các tin nhắn phù hợp với tiêu chí bộ lọc và được nhận giữa lần cuối cùng chức năng này được gọi và bây giờ.
### Thông số
`String`: Khóa bí mật
`String`: Nội dung tin nhắn
### Trả lại
`String`: Trả về một chuỗi đã được mã hóa

### Ví dụ
```javascript
signMessage(
      "0xd61f3aa744725f4f1d0dfcff889e4e2566d762e7b1d4aadf74eed27096d0e5e2",
      "Hello World"
    )
.then(console.log);

```
