# Chuyển chuỗi định dạng UTF-8 về giá trị hexa
```javascript
web3.utils.utf8ToHex(string)
web3.utils.stringToHex(string) // ALIAS
web3.utils.fromUtf8(string) // ALIAS, deprecated
```
Trả về biểu diễn HEX của một chuỗi UTF-8 đã cho.
### Thông số
`string` - `String`: Một chuỗi UTF-8 để chuyển đổi thành chuỗi hex.
### Trả lại
`String`: Chuỗi hex.

### Ví dụ
```javascript
web3.utils.utf8ToHex('I have 100€');
> "0x49206861766520313030e282ac"
```
