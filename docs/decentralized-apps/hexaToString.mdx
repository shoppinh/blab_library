# Chuyển giá trị hexa về số dạng chuỗi
```javascript
web3.utils.hexToNumberString(hex)
```
Trả về biểu diễn số của một giá trị hex đã cho dưới dạng chuỗi.
### Thông số
`hexString` - `String|HEX`: Một chuỗi để băm.
### Trả lại
`String`: Số dưới dạng chuỗi.

### Ví dụ
```javascript

web3.utils.hexToNumberString('0xea');
> "234"
```
