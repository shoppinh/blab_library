# Tạo khóa đối xứng từ mật khẩu

```javascript
web3.shh.generateSymKeyFromPassword(password, [callback])
```
Tạo khóa từ mật khẩu, lưu trữ và trả về ID của nó.
### Thông số
1. `String` - Mật khẩu để tạo khóa SYM từ.
2. `function` - (tùy chọn) gọi lại tùy chọn, trả về một đối tượng lỗi làm tham số đầu tiên và kết quả là thứ hai.
### Trả lại
`Promise<string>|undefined` - Trả về ID chính là Promise hoặc không xác định nếu một cuộc gọi lại được xác định.
### Ví dụ
```javascript
web3.shh.generateSymKeyFromPassword('Never use this password - password!')
.then(console.log);
> "2e57b9ffc2387e18636e0a3d0c56b023264c16e78a2adcba1303cefc685e610f"
```
