# Lấy account từ khóa bí mật

```javascript
getAccountFromPrivateKey(secret)
```
Tạo khóa từ khóa bí mật, lưu trữ và trả về ID của nó.
### Thông số
1. `String` - Khóa bí mật.
### Trả lại
`Object` - Trả về thông tin tài khoản.
### Ví dụ
```javascript
getAccountFromPrivateKey('0x061ae4f6f3ad4e0cf3d944f03c48ba85b141ebf40fc6bb5ec7bba8722bb11397')
.then(console.log);
> {
  "address": "0x68448BE2C20A482310601EbB2ABF4Cd726c1524e",
  "privateKey": "0x061ae4f6f3ad4e0cf3d944f03c48ba85b141ebf40fc6bb5ec7bba8722bb11397",
  "publicKey": "0x04be9a92e617540d79f4680a42d68cced77ec1339d97389a32d2207105a05c5e21d5b7dc89181571ac52f7191aebd01b91939dc14e3ba2d913091d89dd69c55d53"
}
```
