# Chuyển đổi địa chỉ của IBAN (International Bank Account Number) và BBAN (Basic Bank Account Number)
Hàm `Web3.eth.iban` chuyển đổi địa chỉ Ethereum từ và sang IBAN và BBAN.
## Ví dụ Iban
Ví dụ này của Iban.
```javascript
> Iban { _iban: 'XE7338O073KYGTWWZN0F2WZ0R8PX5ZPPZS' }
```
## Iban constructor
```javascript
new web3.eth.Iban(ibanAddress)
```
Tạo đối tượng IBAN với các phương thức chuyển đổi và kiểm tra tính hợp lệ.

Cũng có các chức năng Singleton để chuyển đổi như:

+ Iban.toAddress()
+ Iban.toIban()
+ Iban.fromAddress()
+ Iban.fromBban()
+ Iban.createIndirect()
+ Iban.isValid()
### Thông số
`String`: Địa chỉ IBAN để khởi tạo một ví dụ iban từ.
### Trả lại
`Object` - Trường hợp Iban.

### Ví dụ
```javascript
var iban = new web3.eth.Iban("XE7338O073KYGTWWZN0F2WZ0R8PX5ZPPZS");
> Iban { _iban: 'XE7338O073KYGTWWZN0F2WZ0R8PX5ZPPZS' }
```

## Đến địa chỉ
Hàm tĩnh.
```javascript
web3.eth.Iban.toAddress(ibanAddress)
```
Singleton: Chuyển đổi địa chỉ IBAN trực tiếp thành một địa chỉ Ethereum.
:::note

Phương pháp này cũng tồn tại trong trường hợp IBAN.
:::
### Thông số
`String`: Địa chỉ IBAN để chuyển đổi.
###Trả lại
`String` - Địa chỉ Ethereum.

### Ví dụ
```javascript
web3.eth.Iban.toAddress("XE7338O073KYGTWWZN0F2WZ0R8PX5ZPPZS");
> "0x00c5496aEe77C1bA1f0854206A26DdA82a81D6D8"
```

## đến Iban
Hàm tĩnh.
```javascript
web3.eth.Iban.toIban(address)
```
Singleton: Chuyển đổi địa chỉ Ethereum thành địa chỉ IBAN trực tiếp.

### Thông số
`String`: Địa chỉ Ethereum để chuyển đổi.
### Trả lại
`String` - Địa chỉ IBAN.

### Ví dụ
```javascript
web3.eth.Iban.toIban("0x00c5496aEe77C1bA1f0854206A26DdA82a81D6D8");
> "XE7338O073KYGTWWZN0F2WZ0R8PX5ZPPZS"
```

<ToIBan/>
## Từ địa chỉ
Hàm tĩnh, trả về ví dụ IBAN.
```javascript
web3.eth.Iban.fromAddress(address)
```
Singleton: Converts an Ethereum address to a direct IBAN instance.

### Thông số
`String`: Địa chỉ Ethereum để chuyển đổi.
### Trả lại
`Object` - Trường hợp Iban.
### Ví dụ
```javascript
web3.eth.Iban.fromAddress("0x00c5496aEe77C1bA1f0854206A26DdA82a81D6D8");
> Iban {_iban: "XE7338O073KYGTWWZN0F2WZ0R8PX5ZPPZS"}
```
### Sự thử nghiệm
<FromAddress/>

## Từ ban
```javascript
web3.eth.Iban.fromBban(bbanAddress)
```
Singleton: Chuyển đổi một địa chỉ BBAN thành một ví dụ iban trực tiếp.
### Thông số
`String`: Địa chỉ BBAN để chuyển đổi.
### Returns
`Object` - Ví dụ Iban.
### Ví dụ
```javascript
web3.eth.Iban.fromBban('ETHXREGGAVOFYORK');
> Iban {_iban: "XE7338O073KYGTWWZN0F2WZ0R8PX5ZPPZS"}
```
