# Lấy thông tin từ mạng hiện tại
Gói `Web3-net` cho phép bạn tương tác với các thuộc tính mạng của Node Node.
```javascript
var Net = require('web3-net');

// "Personal.providers.givenProvider" will be set if in an Ethereum supported browser.
var net = new Net(Net.givenProvider || 'ws://some.local-or-remote.node:8546');


// or using the web3 umbrella package

var Web3 = require('web3');
var web3 = new Web3(Web3.givenProvider || 'ws://some.local-or-remote.node:8546');

// -> web3.eth.net
// -> web3.bzz.net
// -> web3.shh.net
```

## Lấy ID
```javascript
web3.eth.net.getId([callback])
web3.bzz.net.getId([callback])
web3.shh.net.getId([callback])
```
Nhận ID mạng hiện tại.
### Thông số
không có

### Trả lại
`Promise` trả về` number`: ID mạng.
### Ví dụ
```javascript
web3.eth.net.getId()
.then(console.log);
> 1
```
### Thử nghiệm
<GetId/>

## Kiêm tra các nút có nghe
```javascript
web3.eth.net.isListening([callback])
web3.bzz.net.isListening([callback])
web3.shh.net.isListening([callback])
```
Kiểm tra xem nút có nghe cho các đồng nghiệp không.
### Thông số
không có

### Trả lại
`Promise` trả lại` boolean`

### Ví dụ
```javascript
web3.eth.net.isListening()
.then(console.log);
> true
```
### Thử nghiệm
<IsListening/>
